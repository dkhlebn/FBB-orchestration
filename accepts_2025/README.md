# Согласия с правилами курса
В этой папке собираются согласия студентов 2025 года с правилами курса, которые получены через PR в этот репозиторий.


## Форма согласия
Вот то, что нужно пропушить в этот репозиторий. Создаёте файл с вашим никнеймом на github `username.yaml`, в нём 
создайте YAML-рзаметку со следующими полями:

```
github_username: <your_username>

first_name: <Your Name>
last_name: <Your Last Name>
repo: <URL or None>
grading: <homeworks OR project>

agreement: |
  Я подтверждаю следующее:
  1. Я не буду списывать решения у других участников курса.
  2. При использовании больших языковых моделей (LLM) или других генеративных ИИ-инструментов
     для выполнения домашних заданий я обязуюсь полностью понимать присланный код и быть
     готовым пояснить, как он работает, почему используется та или иная конструкция,
     а также внести в него изменения по запросу преподавателя.

  Нарушение этих правил может повлечь за собой исключение из курса или 
  аннулирование результатов.

agree_to_rules: yes
```

Пример, как должно в итоге выглядеть. Пусть некий Пётр Лукьянчук хочет закрыть курс посредством 
сдачи всех домашних заданий. Тогда, его файл `peterluce.yaml` выглядит так:
```
github_username: peterluce

first_name: Пётр
last_name: Лукьянчук
repo: https://github.com/peterluce/FBB_infrastruct_HWs
grading: homeworks

agreement: |
  Я подтверждаю следующее:
  1. Я не буду списывать решения у других участников курса.
  2. При использовании больших языковых моделей (LLM) или других генеративных ИИ-инструментов
     для выполнения домашних заданий я обязуюсь полностью понимать присланный код и быть
     готовым пояснить, как он работает, почему используется та или иная конструкция,
     а также внести в него изменения по запросу преподавателя.

  Нарушение этих правил может повлечь за собой исключение из курса или 
  аннулирование результатов.

agree_to_rules: yes

```
