# Проекты

Альтернативой получения зачёта по курсу (если не делать домашки) 
является выполнение проекта в группе по 3-5 человек по созданию 
пайплайна обработки с представлением результатов. Тут будут 
примеры проектов, которые можно реализовать в группе. Отчётность 
по проекту представляет собой Github-репозиторий, в котором 
описан проект и дана инструкция по поднятию и использованию 
сервиса, который решает задачу. Кроме того, в `README` 
репозитория должен быть описан вклад всех участников проекта.

## Требования.

Предположительно, проект должен реализовывать пайплайн обработки 
данных на Airflow, котрый можно запустить из интерфейса streamlit 
автоматически. Внутри пайплайна обязательно должен быть шаг с 
работой в своём кастомном контейнере Docker, который интегрирован 
в конвейер. В Github-репозитории проекта должны быть Github-Actions 
воркфлоу. Использовние API внешних сервисов очень желательно для 
получения данных.

## Примеры возможных проектов.

1. *GEO RNA-seq Differential Expression.*
  Пайплайн обработки данных RNA-seq из NCBI GEO по указанному 
  идентификатору с хотя бы двумя состояниями до таблицы 
  дифф.экспрессии и volcano-plot'а, выводимого в Streamlit.

2. *AlphaFold fetch and feature extraction.*
  Пайплайн обработки (в т.ч. можно отдельно AlphaFold сделать 
  как standalone-контейнер) и предсказания структуры белковой 
  последовательности и вытаскивание её фичей. Визуализация 
  структуры и графиков в Streamlit.

3. *Variant Effect Prediction.*
  Пайплайн обработки VCF-файла с фильтрацией вариантов по указываемым 
  параметрам (задаёте вы) и предсказанием эффекта варианта с помощью 
  ensembl VEP. Дополнительно сделать GWAS и визуализировать с помощью 
  Streamlit.

4. *RNA-protein interaction prediction.*
  Пайплайн обработки данных с сырых данных из ENCODE, доступных по ID, 
  до пиков взаимодействия РНК с белками. На основе этих данных 
  воспроизведение пайплайна машинного обучения и предсказание РНК-белковых 
  взаимодействий. Визуализация в Streamlit.

5. *ChIP-seq motif inference.*
  Пайплайн обработки данных с сырых ChIP-seq ридов из ENCODE или GEO до
  пиков связывания транскрипционных факторов и анализа мотивов связывания 
  белка на ДНК. Визуализация в Streamlit.

## Поддержка проектов

По всем вопросам в реализации проектов пишите преподавателям курса в чат или 
в личные сообщения.
